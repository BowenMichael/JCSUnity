# JCS_TextObject

Holds all type of UI text objects.

## Variables

| Name                  | Description                                      |
|:----------------------|:-------------------------------------------------|
| mTextContainer        | Target text renderer.                            |
| mTextMesh             | Target text renderer. (TMP)                      |

## Functions

| Name | Description              |
|:-----|:-------------------------|
| text | Return text as a string. |
